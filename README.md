------------installation------------

1. Download the script and place it in your scripts folder.
2. Put "ensure muninn-wm" to you `server.cfg`.
3. Edit the config.
4. Restart your server.


------------Configuration------------

1. position - allows to to change the position of the logo.
2. allowoff - allows the use of /watermark command I recommend this being off if you have a big community.
3. framework - choose the frame work you server is. 
4. autohide - enable or disable the auto hide feature. false/true.
5. hideAfter - set the amount of time in seconds to make the logo disappear.

------------Add Image------------

Add you logo in the img folder. Name you logo `logo.png`.

------------End result------------
Now if you followed all these steps your logo should pop up for the amount of time you set. It will open after the player opens the inventory or goes to pause menu.
